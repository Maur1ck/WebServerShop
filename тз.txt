Задание: создать веб-магазин, который будет предоставлять функциональность для покупки товаров онлайн.

Функциональные требования:

Пользователи должны иметь возможность зарегистрироваться на сайте, чтобы создать аккаунт и войти в свой личный кабинет.
Каждый товар должен иметь название, описание, изображение, и цену.
Пользователи должны иметь возможность просмотреть список доступных товаров.
Пользователи должны иметь возможность добавить товары в корзину.
Пользователи должны иметь возможность изменить количество товаров в корзине и удалить товары из корзины.
Пользователи должны иметь возможность оформить заказ и указать адрес доставки.
Пользователи должны иметь возможность просмотреть свою историю заказов и текущий статус заказов.


Нефункциональные требования:

Система должна быть легко масштабируемой, чтобы обеспечить возможность добавления новых функций и обработки большого количества пользователей и товаров.
Система должна быть защищена от несанкционированного доступа и злоупотребления.
Система должна иметь быстрое время ответа для удобства пользователей.
Система должна быть удобной и интуитивно понятной для пользователей и администраторов.


Технологии:

Flask - для создания веб-приложения на Python.
SQLAlchemy - для работы с базой данных.
HTML, CSS и JavaScript - для создания пользовательского интерфейса.
