Задание: Техническое задание на разработку веб-магазина

Цель проекта: Создание веб-магазина для продажи товаров.

Описание проекта: Веб-магазин должен предоставлять пользователям возможность просмотра и покупки товаров, добавления товаров в корзину, оформления заказов, оплаты заказов и просмотра истории заказов.

Основные функции веб-магазина:

Просмотр списка товаров с возможностью фильтрации по категориям, брендам, цене и другим характеристикам;
Просмотр карточки товара с информацией о товаре, его описании, характеристиках, цене, наличии на складе и возможности добавления в корзину;
Добавление товаров в корзину с возможностью изменения их количества и удаления;
Оформление заказа с указанием контактной информации покупателя, адреса доставки и выбором способа оплаты;
Просмотр истории заказов с возможностью отслеживания статуса выполнения заказа и его доставки;
Регистрация пользователей с возможностью сохранения истории заказов, добавления отзывов о товарах и получения персональных скидок и предложений;
Авторизация пользователей с возможностью сохранения корзины и истории заказов;
Поддержка различных методов оплаты заказов (банковские карты, электронные кошельки, наличные и др.);
Поддержка различных способов доставки заказов (курьерская доставка, почта, самовывоз и др.);
Административный интерфейс для управления каталогом товаров, заказами, пользователями, способами оплаты и доставки.


Технологии:

Python - язык программирования для реализации бэкенда;
Flask - фреймворк для создания веб-приложений на языке Python;
SQLAlchemy - ORM для работы с базой данных;
SQLite - СУБД для хранения данных веб-магазина;
HTML, CSS, JavaScript - языки для создания интерфейса пользователя;
Bootstrap - CSS-фреймворк для создания адаптивного дизайна веб-страниц;
jQuery - JavaScript-библиотека для упрощения работы с DOM и AJAX-запросами.


Дополнительные требования:

Разработка веб-магазина должна быть выполнена с использованием методологии Agile;
Разработка должна быть поделена на этапы и каждый этап должен завершен своевременно и быть протестирован перед переходом к следующему этапу;
Веб-магазин должен быть многоязычным, поддерживать не менее трех языков;
Веб-магазин должен иметь возможность работы с несколькими валютами;
Веб-магазин должен иметь возможность добавления и удаления товаров, управления категориями товаров и атрибутами товаров;
Веб-магазин должен иметь систему рейтинга товаров и отзывов покупателей о товарах;
Веб-магазин должен иметь систему корзины, где пользователь может добавлять и удалять товары, а также изменять количество товаров в корзине;
Веб-магазин должен иметь систему оформления заказов, где пользователь может указывать свои данные и адрес доставки, а также выбирать способ доставки и оплаты;
Веб-магазин должен иметь систему авторизации и регистрации пользователей, а также возможность восстановления пароля;
Веб-магазин должен иметь систему управления заказами, где администратор может просматривать и изменять статусы заказов;
